掌握分规则

初始掌握分为 20 分，新内容会立即进入今日复习。

不会：-20 分，下一次安排到明天。

模糊：-8 分，按当前阶段安排。

基本会：+8 分，进入下一阶段。

熟练：+12 分，阶段前进更快。

掌握分达到 90 分及以上，视为已掌握。

复习间隔：1 天、2 天、4 天、7 天、15 天、30 天、60 天。
